# Software Interface and Stack

- **Platform:** Deployed within the enterprise Data Center or Private Cloud. Requires infrastructure for F5 BIG-IP (Virtual Edition or Hardware).
- **Core Technology:** F5 BIG-IP, Lintasarta DekaLLM.
- **Orchestration:** Compatible with standard RAG orchestration frameworks such as LangChain and LlamaIndex.
- **Database:** Interacts with customer-managed Vector Databases (VectorDB). The system stores anonymized embeddings in the VectorDB.
