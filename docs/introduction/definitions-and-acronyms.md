# Definitions and Acronyms

| Term    | Definition |
|---------|------------|
| RAG     | Retrieval-Augmented Generation. An architecture that grounds large language model (LLM) responses in specific knowledge bases. |
| GenAI   | Generative Artificial Intelligence. AI capable of generating text, images, or other data. |
| LLM     | Large Language Model. A type of AI model designed to understand and generate human-like text. |
| LLmaaS  | LLM-as-a-Service. A cloud-based service providing access to large language models. |
| PDP     | Personal Data Protection. Refers to Indonesia's data privacy law (UU No. 27 Tahun 2022). |
| PII     | Personally Identifiable Information. Any data that could potentially identify a specific individual. |
| F5 BIG-IP | A multi-purpose networking device from F5 that can manage traffic, provide security, and optimize application performance. |
| MCN     | Secure Multi-Cloud Networking. A framework for providing secure and consistent network connectivity across multiple cloud environments. |
| WAF     | Web Application Firewall. A security solution that protects web applications from various attacks. |
| SIEM    | Security Information and Event Management. A solution that collects and analyzes security data to detect threats. |
| VectorDB | Vector Database. A database designed to store and query high-dimensional vector embeddings, used in RAG for similarity searches. |
